# 📝 Flask Blog

A full-stack blog application built using Flask that enables users to register, log in securely, manage profiles, and create, edit, and delete blog posts. The project demonstrates backend web development concepts, user authentication, database integration, and RESTful application design using Flask and SQLite.

---

## 📖 Project Overview

Flask Blog is a web application designed to provide a complete blogging platform where authenticated users can publish and manage blog posts. It showcases core Flask concepts such as authentication, routing, form validation, database operations, and template rendering while following a modular and scalable project structure.

---

## ✨ Features

- User registration and secure authentication
- User login and logout
- Profile management
- Create, read, update, and delete (CRUD) blog posts
- Password hashing for secure authentication
- SQLite database integration
- Flash messages for user feedback
- Responsive user interface
- RESTful routing using Flask
